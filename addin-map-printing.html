Add-in Tools for Map Printing

C#ArcGIS Pro SDKArcGIS ProPythonAutomation

This project focuses on the migration and enhancement of robust C# ArcGIS Pro Add-ins designed to automate map printing and streamlining planning scheme amendment workflows. It replaces legacy tools, ensuring seamless compatibility and integration within the current ArcGIS Pro framework.

Key Features & Development:

  • Migrated complex ArcObjects-based tools from ArcMap to the contemporary ArcGIS Pro SDK utilizing C# and .NET frameworks.
  • Automated map production for gazetting purposes, greatly accelerating the spatial document layout generation.
  • Integrated Python scripting via ArcPy to complement automation tasks handling large-scale geodatabase operations.
  • Enhanced user interface capabilities with custom ribbons and responsive dockpanes for a better user experience for GIS operators.
  • Ensured robust quality control mechanisms within the printing process for planning scheme amendments.

Impact:

The development of these tailored add-ins eliminated significant manual mapping effort, driving higher accuracy and faster turnaround times on planning scheme processes, providing end users with a robust, scalable toolset.
