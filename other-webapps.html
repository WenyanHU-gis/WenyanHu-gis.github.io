Renewables Summary Page

A high-visibility dashboard for the Department of Transport and Planning, summarizing Victoria's renewable energy generation facilities by status and type.

Key Functions:

  • Area Search: Quickly find and zoom to specific project areas.
  • Dynamic Summaries: Automatic tabulation of capacity (MW) across operating, approved, and Lodged projects.
  • Data Export: Functionality to download data tables for offline analysis.
  • Layer Management: Interactive layer list to toggle specific project visibility.
  • Categorical Tabs: Specific visualization for Solar, Wind, Battery, and Bioenergy datasets.
Technologies: ArcGIS API for JavaScript, Spatial Data Governance

Activity Centres Program

A public-facing web map supporting the Victorian Government's initiative to plan for more than 300,000 new homes across transit-connected locations by 2051. The interactive platform provides spatial context for the initial 10 pilot activity centres (including Broadmeadows, Chadstone, and Frankston), helping communities engage with proposed design requirements and catchment boundaries.

Key Functions:

  • Interactive Mapping: Explore proposed building heights, core commercial areas, and low-rise catchment zones in key suburban centres.
  • Community Engagement: Connects detailed spatial data with public consultation initiatives as part of Victoria's Housing Statement.
  • Dynamic Data Visualization: Seamless rendering of complex planning controls, enabling users to easily digest proposed changes to their neighbourhoods.
Technologies: TypeScript, ArcGIS API for JavaScript

Public Transport Accessibility Level (PTAL)

An interactive mapping instrument visualizing the density and quality of the public transport network across Victoria. Assists urban planners, developers, and the community in measuring access to transport services.

Key Functions:

  • Accessibility Scoring: Displays calculated PTAL indexes reflecting distance to stops and service frequency.
  • Strategic Planning Context: Directly supports policy decisions (like the Activity Centres Program) by highlighting transit-rich areas.
  • Fact Sheet Integration: Embeds references to the PTAL Fact Sheet for transparent methodology.
Technologies: JavaScript, Spatial Analysis, ArcGIS API for JavaScript

Communication and Community Engagement (CCE) Map

A robust spatial platform designed to facilitate place-based community engagement. It equips users with advanced interactive tools, supporting both internal planning analysis and public data interaction.

Key Functions:

  • Custom Data Upload: Empowers users to temporarily upload their own Shapefiles (SHP) into the map for contextual visual analysis.
  • Advanced Querying: Allows users to perform complex spatial selections based on overlapping geometries and attribute filtering.
  • Data Export Capabilities: Users can effortlessly extract and export queried datasets for offline reporting.
Technologies: JavaScript, ArcGIS API for JavaScript, Custom Widgets
